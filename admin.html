🔐 Введіть GitHub-токен

Адмінка завантажує фото товарів напряму до репозиторію GitHub. Для цього потрібен Personal Access Token із доступом до запису у catalog-full.

Як отримати токен (простіше — Classic):
  1. Відкрий github.com/settings/tokens/new
  2. Note: ltex-admin
  3. Expiration: 1 year (або No expiration)
  4. Select scopes: постав ✅ на repo
  5. Натисни Generate token внизу → скопіюй (ghp_…)
Або Fine-grained (з обмеженим доступом)
  1. Відкрий settings/personal-access-tokens/new
  2. Name: ltex-admin, Expiration: 1 year
  3. Repository access → Only select repositories → обери catalog-full
  4. Прокрути ВНИЗ → секція Permissions → Repository permissions
  5. Знайди Contents → справа dropdown → Read and write
  6. Generate token → скопіюй (github_pat_…)

Токен зберігається тільки на цьому пристрої (localStorage), не передається на інші сервери.
